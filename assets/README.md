# Danh Sách Ảnh Món Ăn - Nhà Hàng KING

## Hướng Dẫn Thay Thế Ảnh
- Các file placeholder hiện tại là SVG/placeholder URLs
- Để thay thế, upload ảnh thực vào folder `assets/`
- Tên file phải khớp với list dưới đây (hoặc update `menu.json`)
- Format: JPG, PNG (khuyến nghị JPG để nhanh load)
- Kích thước: 300x250px (hoặc ratio 6:5)

## Cách Upload Ảnh
1. Chụp/tải ảnh thực của từng món
2. Resize về 300x250px
3. Đặt tên file theo danh sách trên
4. Copy vào folder `assets/`
5. Hệ thống sẽ tự động load ảnh mới

## Lưu Ý
- Ảnh phải có license cho phép sử dụng
- Nên dùng ảnh có chất lượng cao (72dpi tối thiểu)
- File quá to sẽ làm website chậm (nén ảnh nếu cần)
